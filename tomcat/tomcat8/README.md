# Tomcat 8
Contenedor Docker basado en la última versión de Ubuntu disponible en las imágenes Docker oficiales, Oracle JDK 8 y Tomcat 8.

# Descripción
Incluye:
- Oracle JDK 1.8
- Tomcat 8.0.35
- curl

# Puertos
Puertos expuestos:
- 8080, el puerto por defecto de Tomcat.
